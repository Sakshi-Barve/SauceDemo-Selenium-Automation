# SauceDemo-Selenium-Automation
Project Overview-
This repository contains a Selenium-based automation suite for the Sauce Demo e-commerce application. Currently, it includes automated test cases for valid login, invalid login, and product filtering.

Features-
Automated login tests
Product sorting and filtering automation
Checkout and payment process testing

Setup Instructions
1. Clone the repository: git clone https://github.com/Sakshi-Barve/SauceDemo-Selenium-Automation.git  
2. Navigate to the project directory: cd SauceDemo-Selenium-Automation  
3. Install dependencies: mvn clean install  
4. Run the tests: mvn test  

Usage-
After setting up, you can run the test suite using Maven. The results will be available in the target directory.
